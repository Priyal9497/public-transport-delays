# Public Transport Delays — EDA Project

## Overview
An exploratory data analysis project examining how weather conditions,
events, holidays, and peak hours affect public transport delays.

## Objectives
1. Analyze delays caused by weather conditions
2. Identify delay-prone routes and stations
3. Compare delays on holidays, weekends, and weekdays
4. Perform statistical tests (T-test, Chi-square, Shapiro-Wilk)
5. Build a Linear Regression model to predict arrival delay

## Tools & Libraries
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Scipy, Scikit-learn, Statsmodels

## Dataset
Public Transport Delays with Weather & Events dataset
